\doxysubsection{Detailed Description}
Rappresenta una transazione di prestito all\textquotesingle{}interno della biblioteca. 


\begin{DoxyItemize}
\item Questa classe associa un \doxylink{classmodel_1_1_utente}{Utente} a un \doxylink{classmodel_1_1_libro}{Libro}, tracciando la data di inizio, la data di scadenza prevista e lo stato corrente del prestito (attivo o concluso), la variabile prestito attivo evidenzia velocemente i prestiti in ritardo.
\item \begin{DoxyAuthor}{Author}
