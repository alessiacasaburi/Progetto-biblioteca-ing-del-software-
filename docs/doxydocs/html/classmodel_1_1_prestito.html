Detailed Description

Rappresenta una transazione di prestito all'interno della biblioteca.

  • Questa classe associa un Utente a un Libro, tracciando la data di inizio, la data di scadenza prevista e lo stato corrente del prestito (attivo o concluso), la variabile prestito attivo evidenzia velocemente i prestiti in ritardo.
