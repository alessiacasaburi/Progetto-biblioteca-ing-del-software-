Detailed Description

Interfaccia generica per la gestione delle entità del sistema.

  • Definisce i metodi fondamentali (CRUD) che ogni classe di gestione (es. GestoreLibri, GestoreUtenti) deve implementare.
  • Template Parameters
    TIl tipo di oggetto gestito dall'implementazione (es. Libro, Utente, Prestito).
