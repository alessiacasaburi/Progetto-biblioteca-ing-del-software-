Detailed Description

Rappresenta un utente registrato nella biblioteca.

  • Questa classe gestisce i dati anagrafici dell'utente (nome, cognome, matricola, email) e mantiene la lista dei prestiti attualmente attivi associati all'utente stesso.
  • Author
    Alessandro
